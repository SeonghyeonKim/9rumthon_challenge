# 문제 1. 운동 중독 플레이어

예상 티어: 브론즈 5
예상 태그: 수학, 사칙연산

<br>

## 문제 설명

1RM = W * (1 + R / 30)
W, R을 입력 받았을 때, 1RM을 구하는 문제.

1 <= W <= 1,000,000
1 <= R <= 100
정답은 소숫점 이하의 값을 버린 뒤 출력한다.

<br>

## 문제 해결

주어진 수식을 그대로 작성하면 된다.

```cpp
cout << int(W * (1+R*1.0/30));
```
